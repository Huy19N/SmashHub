# 非機能要件一覧

| id | category | requirement | targetValue | priority |
| --- | --- | --- | --- | --- |
| NFR-001 | 性能 | 画面応答速度 ※要確認 | 通常画面（顧客一覧、案件登録等）はボタン押下から3秒以内、ダッシュボード（FR-017）などの大量データ集計を伴う画面は5秒以内で処理を完了すること ※要確認 | 高 |
| NFR-002 | 性能 | バッチ処理時間 ※要確認 | 売上データ自動抽出バッチ（FR-013）は、システムへの負荷および他処理への影響を避けるため、夜間帯（2:00〜5:00）に起動し、最大30分以内に完了すること。なお、バッチの稼働周期・詳細仕様は未定義のため要確認。 ※要確認 | 高 |
| NFR-003 | 性能 | 同時接続ユーザー数 ※要確認 | 全社での利用（営業・管理等）を想定し、最大同時接続50ユーザーが同時に操作した場合でも極端な遅延が発生しないこと。同時接続のピーク予測値は未定義。 ※要確認 | 中 |
| NFR-004 | 可用性 | 目標稼働率・システム提供時間 ※要確認 | 平日および土日祝日の24時間365日サービス提供を原則とし、年間稼働率99.5%（メンテナンス時間を除く）以上を維持すること ※要確認 | 高 |
| NFR-005 | 可用性 | バックアップ・データ復旧 ※要確認 | データベースは日次で自動フルバックアップを取得し、世代管理を30世代分行うこと。RPO（目標復旧時点）は最大24時間前、RTO（目標復旧時間）は障害検知後24時間以内。 ※要確認 | 高 |
| NFR-006 | 可用性 | 災害復旧（DR）および事業継続（BCP）対策 ※要確認 | クラウドサービスにおける別リージョンへのデータバックアップの保管。地理的冗長化や自動フェイルオーバーを伴うDRサイト構築の有無は未定義。 ※要確認 | 低 |
| NFR-007 | セキュリティ | 通信およびデータの暗号化 ※要確認 | インターネットを通過するすべての通信（HTTP）はSSL/TLSによって暗号化（HTTPS）すること。データベースに保管されるパスワードや重要データは一方向ハッシュ化等により暗号化すること ※要確認 | 高 |
| NFR-008 | セキュリティ | 認証・アクセス制限（FR-020関連） ※要確認 | IDおよびパスワード（8文字以上、英数字混合必須など）による認証。管理者、マネージャー、一般ユーザーのロールによるアクセス制限の実施。多要素認証（MFA）やSSO（シングルサインオン）の導入有無は未定義。 ※要確認 | 高 |
| NFR-009 | セキュリティ | アクセスログおよび操作履歴の保持 ※要確認 | システムログイン・ログアウトログ、データの作成・更新・削除を伴う主要な操作（FR-002, FR-007, FR-013等）のログ。最低1年間は安全な領域にログを保持すること。監査対応の有無は未定義。 ※要確認 | 高 |
| NFR-010 | 運用 | システム監視および障害検知・通知 ※要確認 | サーバーリソース（CPU、メモリ、ストレージ）の監視を自動で行い、閾値（例: 80%）超過時に管理者へ自動通知すること。死活監視（Ping等）によるサービス停止検知時は即時に通報すること。 ※要確認 | 中 |
| NFR-011 | 運用 | 定期メンテナンスウィンドウ ※要確認 | 計画メンテナンス（バグ修正やアップデート適用等）は、全社業務時間外（例：土曜日 22:00〜翌日6:00等）に行う。ユーザーへの事前通知機能または代替表示を行うこと。 ※要確認 | 中 |
| NFR-012 | 移行 | 初期データ移行の実施 ※要確認 | 各部門で個別に運用されているExcelや既存の顧客・売上データ等のデータ移行方針（移行ツール、手動入力、一括取込）およびフォーマットの策定。既存システムの正確なデータレイアウトが未提供のため移行定義は一部未定義。 ※要確認 | 高 |
| NFR-013 | 拡張性 | データボリュームの増加対応 ※要確認 | 顧客、案件、売上データの登録件数の増加に対し、少なくとも5年間分のデータ（累積データ容量100GB想定）をパフォーマンス劣化を伴うことなく格納・運用可能であること ※要確認 | 中 |
| NFR-014 | 拡張性 | 外部システム連携インターフェース（API） ※要確認 | 他システム（会計システム、既存基幹システム等）との連携を見据え、拡張が可能なWeb API形式によるデータ入出力構造を考慮した設計を行うこと。具体的な連携仕様は技術制約が未提供のため未定義。 ※要確認 | 中 |
